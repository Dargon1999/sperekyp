# MoneyTracker Pro Website (Cyber Ledger Edition)

Профессиональный лендинг и админ-панель для системы учета MoneyTracker Pro v9.3.

## 🚀 Как запустить

### 1. Локальный запуск (через Python)
Это рекомендуемый способ для тестирования всех функций (включая загрузку файлов).
1. Убедитесь, что у вас установлен Python.
2. Откройте терминал в папке `website`.
3. Установите зависимости: `pip install -r requirements.txt`
4. Запустите сервер: `python app.py`
5. Откройте в браузере: `http://localhost:5000`

### 2. Быстрый просмотр (Demo Mode)
Вы можете просто открыть `index.html` в браузере. Сайт будет работать в демо-режиме:
- Контент будет загружен из встроенного кэша.
- Авторизация `admin / admin123` будет работать локально.
- *Загрузка файлов на диск и сохранение изменений в файлы в этом режиме невозможны.*

## 🛠 Админ-панель
- **Вход**: Клик по логотипу в шапке/футере или горячие клавиши `Ctrl + Shift + A`.
- **Данные**: `admin` / `admin123`.
- **Функционал**: Редактирование текста, управление медиа-библиотекой (Drag-and-Drop), деплой обновлений ПО.

## ☁️ Деплой на Render.com
1. Создайте новый **Web Service** на Render.
2. Подключите ваш репозиторий GitHub.
3. Настройки:
   - **Build Command**: `pip install -r requirements.txt`
   - **Start Command**: `gunicorn wsgi:app`
4. Добавьте переменную окружения `JWT_SECRET_KEY` (любая длинная строка).
